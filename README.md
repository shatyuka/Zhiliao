## 功能

- 去启动页广告
- 去信息流广告
- 去回答列表广告
- 去评论广告
- 去分享广告
- 去回答底部广告
- 过滤视频
- 去信息流会员推荐
- 去回答圈子
- 去商品推荐
- 直接打开外部链接
- 禁止切换色彩模式
- 移除直播按钮
- 不显示小红点
- 隐藏会员卡片
- 隐藏会员按钮
- 隐藏热点通知
- 左右划切换回答
- 移除下一个回答按钮
- 自定义过滤
